# 💪 Antonio Júnior - Consultoria Online

Landing page desenvolvida para apresentação de serviços de consultoria fitness personalizada.

## 🚀 Acesse o site

👉 https://antonio-junior-consultoria-online.vercel.app/

## 📸 Sobre o projeto

Este projeto é uma landing page moderna focada em conversão, com objetivo de apresentar os serviços de personal trainer online, destacando resultados, depoimentos e chamada para ação.

## 🛠️ Tecnologias utilizadas

* React
* TypeScript
* Vite
* CSS

## ⚙️ Como rodar localmente

```bash
# Instalar dependências
npm install

# Rodar projeto
npm run dev
```

## 📦 Build

```bash
npm run build
```

## 🌐 Deploy

O projeto está hospedado na Vercel com deploy automático via GitHub.
